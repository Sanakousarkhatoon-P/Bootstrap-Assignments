All Bootstrap colors are available in the SCSS/ variables. SCSS file as SASS variables and a SASS map. For each of these variables, Bootstrap does not create text or background color classes in order to reduce file sizes. Instead, they decide on a smaller subset of these hues to create a themed palette.
